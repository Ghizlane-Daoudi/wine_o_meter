The wine-o-meter API

Welcome to the wine-o-meter API.

Endpoints

  • /predict, method POST

    Predict the quality of the wine with a score from 0 to 10 based on physicochemical samples.

    The expected input is a JSON with an input key followed by an array of inputs. Each inputs is an array of samples in the following order:

    1. fixed acidity
    2. volatile acidity
    3. citric acid
    4. residual sugar
    5. chlorides
    6. free sulfur dioxide
    7. total sulfur dioxide
    8. density
    9. pH
    10. sulphates
    11. alcohol

    JSON input example with one single input (one single wine): 

                
        {
          "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]
        }

    JSON input example with multiple inputs (a list of two or more wines): 

                
        {
          "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [5.0, 0.27, 0.36, 10.7, 0.045, 45.0, 210.0, 1.001, 3.0, 0.45, 5.8]]
        }

    Request example using curl: 

                
        curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' https://wineometer-byghizlane.herokuapp.com/predict

    Expected output: 

                
        {
          "prediction": [6, 6]
        }
